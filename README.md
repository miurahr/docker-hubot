docker-hubot-lingr
===================
Dockerfile to build image which is installed hubot.  

## Usage

```
$ sudo docker pull miurahr/hubot-lingr
$ sudo docker run -d -p 22 miurahr/hubot-lingr
```

## Contents

The image contains:

- hubot 2.6.0
- hubot-lingr
- hubot-trello
